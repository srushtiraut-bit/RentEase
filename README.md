# RentEase

A full-stack rental management web application built using the MERN stack.  
RentEase helps users browse rental properties, manage listings, and streamline the rental process with a modern and responsive interface.

---

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Clone Repository

```bash
git clone https://github.com/srushtiraut-bit/RentEase.git
cd RentEase
```

---

## Frontend Setup

```bash
cd RentEase-frontend
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

---

## Backend Setup

```bash
cd RentEase-backend
npm install
npm start
```

Backend runs on:

```text
http://localhost:5000
```

---

## Environment Variables

Create `.env` files in both frontend and backend folders.

### Backend `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
